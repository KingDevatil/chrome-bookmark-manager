功能强大的 Chrome 浏览器收藏夹管理插件，提供侧边栏快速访问、批量管理和 WebDAV 自动备份等功能。

## ✨ 功能特性

### 🎯 核心功能

- **侧边栏快速访问** - 点击浏览器工具栏图标即可在侧边栏快速查看和访问书签
- **全页面管理器** - 提供完整的书签管理界面，支持列表和网格两种视图模式
- **文件夹树导航** - 左侧文件夹树支持折叠/展开，快速定位目标文件夹
- **批量操作** - 支持 Shift 多选、全选、批量删除等功能
- **拖拽排序** - 支持拖拽书签调整顺序或移动到其他文件夹
- **常用目录** - 自动统计最近访问最频繁的链接，支持自定义时间范围、显示数量、置顶和黑名单

### 🎨 界面优化

- **深色模式** - 支持深色/浅色主题切换，保护视力
- **响应式设计** - 自适应不同屏幕尺寸
- **自定义布局** - 可调整书签高度、目录缩进和书签缩进宽度
- **现代化 UI** - 简洁美观的 Material Design 风格
- **右键菜单** - 书签和文件夹支持右键删除、编辑操作

### 💾 数据备份

- **WebDAV 支持** - 支持将书签备份到任意 WebDAV 服务器
- **自动备份** - 可设置定时自动备份，支持启动时自动备份
- **导入导出** - 支持 JSON 格式的书签导入导出
- **合并/覆盖模式** - 灵活的恢复选项
- **标签备份** - 支持独立导出/导入书签标签数据
- **数据清理** - 自动检测并清理已删除书签的孤立标签

### 🏷️ 标签管理

- **标签分类** - 为书签添加自定义标签，便于分类管理
- **标签分组** - 创建分组来组织标签，支持创建、删除、重命名分组
- **快速添加** - 书签详情面板提供标签选择器，点击即可添加/移除标签
- **标签搜索** - 搜索时自动匹配标签，快速找到相关书签
- **标签备份** - 标签和分组数据支持本地导出/导入和 WebDAV 备份

### 📋 搜索功能

- **模糊搜索** - 同时搜索书签标题、网址和标签
- **实时过滤** - 输入即搜索，快速定位目标书签

### ⚡ 便捷操作

- **快捷键支持**
  - `Ctrl+A` / `Cmd+A` - 全选
  - `Delete` - 删除选中书签
  - `Esc` - 取消选择
  - `Shift+ 点击` - 批量选择范围
- **双击操作**
  - 双击书签 - 在新标签页打开链接
  - 双击文件夹 - 快速导航到子文件夹
- **面包屑导航** - 清晰的路径指示和快速跳转
- **文件夹选择器** - 编辑书签时可移动到其他文件夹，支持树形结构展开/折叠

## 🚀 使用指南

### 侧边栏使用

1. 点击浏览器工具栏的扩展图标
2. 侧边栏会显示书签树
3. 点击书签即可在新标签页打开
4. 点击右上角菜单按钮可切换深色模式或进入设置
5. 右键点击书签或文件夹可进行删除、编辑操作
6. 拖拽书签可调整顺序或移动到其他文件夹

### 常用目录

1. 点击侧边栏菜单按钮，开启"启用常用"开关
2. 侧边栏顶部会显示"⭐ 常用"文件夹
3. 自动统计最近访问最频繁的链接
4. 右键点击常用链接可快速添加到书签、置顶或屏蔽域名
5. 置顶的链接固定在常用文件夹顶部，不受热度刷新影响
6. 在设置中可配置时间范围（3/7/10 天）和显示数量（5-15 个）

### 全页面管理

1. 点击侧边栏底部的 **管理器** 按钮
2. 或直接访问 `chrome-extension://[扩展 ID]/manager/manager.html`
3. 左侧为文件夹树，右侧为书签列表
4. 支持添加书签、新建文件夹、删除、拖拽等操作
5. 点击书签右侧编辑按钮可修改名称、URL 和移动文件夹

### WebDAV 备份设置

1. 打开设置页面
2. 进入 **WebDAV 同步** 标签
3. 填写服务器地址、用户名和密码
4. 可选择是否启用自动备份
5. 点击 **立即备份** 测试连接

### 布局自定义

1. 打开设置页面
2. 进入 **布局** 标签
3. 调整书签按钮高度（24-48px）
4. 调整目录缩进宽度（5-20px）
5. 调整书签缩进宽度（5-20px）

### 标签管理

1. **添加标签**
   - 打开书签管理器
   - 点击任意书签，右侧显示详情面板
   - 在标签输入框输入标签名后按回车
   - 或选中多个书签后点击「添加标签」按钮批量添加

2. **删除标签**
   - 点击书签列表或详情页中标签上的 × 按钮

3. **标签搜索**
   - 在搜索框输入 `#标签名`
   - 支持模糊匹配（不区分大小写）
   - 只显示当前目录及子目录内的结果

4. **标签总览**
   - 打开设置页面
   - 点击左侧菜单的 **🏷️ 标签总览**
   - 查看所有标签及书签数量
   - 点击标签卡片查看该标签下的所有书签

5. **标签备份**
   - 打开设置页面
   - 进入 **备份与恢复** 标签
   - 滚动到 **标签备份** 区域
   - 点击 **导出标签** 备份所有标签数据
   - 点击 **导入标签** 从 JSON 文件恢复标签
   - 点击 **开始检测** 检查孤立标签
   - 点击 **清理无效标签** 删除已删除书签的标签数据

6. **WebDAV 备份**
   - 使用 WebDAV 备份时自动包含标签数据
   - 恢复时自动检测并恢复标签
   - 兼容旧版本无标签数据的备份文件

## 📝 常见问题

### Q: 为什么侧边栏打不开？
A: 请确保已授予扩展必要的权限，并尝试重新加载扩展。

### Q: WebDAV 备份失败？
A: 检查服务器地址、用户名和密码是否正确，确保网络连接正常。

### Q: 书签同步有问题？
A: 本扩展使用 Chrome 本地书签 API，书签会自动与 Chrome 账户同步。WebDAV 备份是额外的数据保护措施。
