# Chrome Bookmark Manager

功能强大的 Chrome 浏览器收藏夹管理插件，提供侧边栏快速访问、批量管理和 WebDAV 自动备份等功能。

![Version](https://img.shields.io/badge/version-1.0.0-blue.svg)
![Manifest](https://img.shields.io/badge/manifest-v3-green.svg)

## ✨ 功能特性

### 🎯 核心功能

- **侧边栏快速访问** - 点击浏览器工具栏图标即可在侧边栏快速查看和访问书签
- **全页面管理器** - 提供完整的书签管理界面，支持列表和网格两种视图模式
- **文件夹树导航** - 左侧文件夹树支持折叠/展开，快速定位目标文件夹
- **批量操作** - 支持 Shift 多选、全选、批量删除等功能
- **拖拽排序** - 支持拖拽书签调整顺序或移动到其他文件夹

### 🎨 界面优化

- **深色模式** - 支持深色/浅色主题切换，保护视力
- **响应式设计** - 自适应不同屏幕尺寸
- **自定义布局** - 可调整书签高度和目录缩进宽度
- **现代化 UI** - 简洁美观的 Material Design 风格

### 💾 数据备份

- **WebDAV 支持** - 支持将书签备份到任意 WebDAV 服务器
- **自动备份** - 可设置定时自动备份，支持启动时自动备份
- **导入导出** - 支持 JSON 格式的书签导入导出
- **合并/覆盖模式** - 灵活的恢复选项

### ⚡ 便捷操作

- **快捷键支持**
  - `Ctrl+A` / `Cmd+A` - 全选
  - `Delete` - 删除选中书签
  - `Esc` - 取消选择
  - `Shift+ 点击` - 批量选择范围
- **双击进入文件夹** - 快速导航到子文件夹
- **面包屑导航** - 清晰的路径指示和快速跳转

### 方法二：开发者模式安装

1. 下载本项目代码
2. 打开 Chrome 浏览器，访问 `chrome://extensions/`
3. 开启右上角的 **开发者模式**
4. 点击 **加载已解压的扩展程序**
5. 选择本项目所在的文件夹
6. 扩展安装完成，会在浏览器工具栏显示图标

## 🚀 使用指南

### 侧边栏使用

1. 点击浏览器工具栏的扩展图标
2. 侧边栏会显示书签树
3. 点击书签即可在新标签页打开
4. 点击右上角菜单按钮可切换深色模式或进入设置

### 全页面管理

1. 点击侧边栏底部的 **管理器** 按钮
2. 或直接访问 `chrome-extension://[扩展 ID]/manager/manager.html`
3. 左侧为文件夹树，右侧为书签列表
4. 支持添加书签、新建文件夹、删除、拖拽等操作

### WebDAV 备份设置

1. 打开设置页面
2. 进入 **WebDAV 同步** 标签
3. 填写服务器地址、用户名和密码
4. 可选择是否启用自动备份
5. 点击 **立即备份** 测试连接

### 布局自定义

1. 打开设置页面
2. 进入 **布局** 标签
3. 调整书签按钮高度（24-48px）
4. 调整目录缩进宽度（5-32px）

## 🔧 技术栈

- **纯原生 JavaScript** - 无需任何框架，轻量高效

## 📝 常见问题

### Q: 为什么侧边栏打不开？
A: 请确保已授予扩展必要的权限，并尝试重新加载扩展。

### Q: WebDAV 备份失败？
A: 检查服务器地址、用户名和密码是否正确，确保网络连接正常。

### Q: 书签同步有问题？
A: 本扩展使用 Chrome 本地书签 API，书签会自动与 Chrome 账户同步。WebDAV 备份是额外的数据保护措施。
