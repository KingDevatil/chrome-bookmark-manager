操作说明

🚀 快速开始

侧边栏

  • 打开侧边栏：点击浏览器工具栏中的扩展图标
  • 打开书签：点击书签项即可在新标签页中打开
  • 展开/折叠文件夹：点击文件夹或左侧的展开箭头
  • 搜索书签：在搜索框中输入关键词进行搜索
  • 切换主题：点击菜单按钮 → 深色模式开关
  • 右键菜单：书签和文件夹支持右键删除、编辑操作
  • 拖拽排序：可拖拽书签调整顺序
  • 拖拽移动：可拖拽书签到文件夹中移动位置

常用目录

  • 启用常用目录：点击菜单按钮 → 启用常用开关
  • 动态统计：自动统计最近访问最频繁的链接
  • 快速添加书签：右键点击常用链接可快速添加到书签
  • 黑名单功能：右键屏蔽特定域名，不再出现在常用列表中
  • 自动刷新：打开侧边栏时自动更新，每 5 分钟自动刷新

📚 管理器

导航

  • 浏览文件夹：点击左侧文件夹树中的文件夹
  • 折叠文件夹：点击文件夹前的 ▶ 箭头图标
  • 面包屑导航：点击顶部路径可快速跳转

书签操作

  • 添加书签：点击「+ 添加书签」按钮
  • 新建文件夹：点击「+ 新建文件夹」按钮，可选择存放位置
  • 打开书签：双击书签项可在新标签页中打开链接
  • 选择书签：单击书签项（不需要点击复选框）
  • 批量选择：按住 Shift 键点击可选择范围内的所有书签
  • 全选：按 Ctrl+A (Mac: Cmd+A)
  • 取消选择：按 Esc 键
  • 删除书签：点击书签右侧的删除按钮，或选中后按 Delete 键
  • 编辑书签：点击书签右侧的编辑按钮，可修改名称、URL 和移动文件夹

拖拽操作

  • 排序书签：拖拽书签行可调整顺序
  • 移动到文件夹：将书签拖拽到左侧文件夹树上

视图切换

  • 列表视图：显示详细信息，适合管理
  • 网格视图：卡片式布局，适合浏览

🏷️ 标签管理

书签标签

  • 添加标签：在书签详情面板输入标签名后按回车
  • 批量添加标签：选中多个书签后点击「添加标签」按钮
  • 删除标签：点击标签上的 × 按钮即可删除
  • 标签显示：书签列表和详情页都会显示标签
  • 标签搜索：在搜索框输入 #标签名 可搜索包含该标签的所有书签

标签总览

  • 查看所有标签：设置 → 标签总览，显示所有标签及书签数量
  • 查看标签详情：点击标签卡片可查看该标签下的所有书签
  • 打开书签：点击书签列表中的书签可在新窗口打开

标签备份

  • WebDAV 备份：使用 WebDAV 备份时自动包含标签数据
  • 独立导出：设置 → 备份与恢复 → 导出标签，导出为 JSON 文件
  • 独立导入：从 JSON 文件导入标签数据（合并到现有数据）
  • 数据清理：检测并清理已删除书签的孤立标签

⚙️ 设置

WebDAV 同步

  • 配置 WebDAV 服务器地址和认证信息
  • 支持自动定时备份
  • 支持启动时自动备份
  • 备份数据包含书签和标签

备份与恢复

  • 合并模式：将备份与现有书签合并，保留现有书签
  • 覆盖模式：删除所有现有书签，用备份替换（谨慎使用）
  • 标签备份：独立导出或导入标签数据
  • 数据清理：检测并清理孤立标签

常用目录设置

  • 启用/禁用：开启或关闭常用目录功能
  • 时间范围：选择统计周期（3/7/10 天）
  • 显示数量：设置显示的常用链接数量（5-15 个）
  • 黑名单管理：查看和管理已屏蔽的域名列表
  • 置顶管理：查看和管理已置顶的链接，置顶链接固定在常用文件夹顶部

布局设置

  • 书签按钮高度：调整侧边栏中书签项的高度 (24-48px)
  • 目录缩进宽度：调整树形目录中子文件夹的缩进 (5-20px)
  • 书签缩进宽度：调整书签相对于目录的缩进 (5-20px)

⌨️ 快捷键

Ctrl+A / Cmd+A 全选当前文件夹下的所有书签
Delete 删除选中的书签
Esc 取消选择
Shift+点击 批量选择范围内的书签
选择标签添加到分组
